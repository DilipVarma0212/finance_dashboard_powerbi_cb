# Finance Data Analysis

*This project focuses on analyzing day-to-day personal financial data to monitor income, expenses, and savings patterns.*


## Table of contents


### Overview

The dashboard provides actionable insights to improve financial planning and decision-making using interactive visualizations.



### Business Problem:

Personal financial management lacks visibility and control, leading to high expenses (78%) and low savings (22%). There is a need to identify major spending areas, inefficient saving patterns, and monthly fluctuations in order to optimize budgeting and improve overall financial health.



### Data Source

- The primary dataset used for this analysis is the "finance-data.xlsx" file, containing detailed information about each sale made by the store.



### Tools

- Excel (Dataset)
- Power BI (Dashboard Creation)
- Data Cleaning (Power Query)
- Data Visualization Techniques
- GitHub



### Data Cleaning Process

1. Ensured data accuracy for analysis
2. Removed null values and duplicates
3. Fixed inconsistent and incorrect data
4. Applied filters and slicers for better user experience



### Business Questions

- What is the total income?
- What is the percentage split between expenses and savings?
- Which category contributes the highest and lowest expenses?
- Which components are best/worst for savings?
- Which months show high expenses vs high savings trends?



 ### Results

- The total income is ₹2,000,000
- The percentage split between expenses and savings is 72% and 22% resp.
- House rent is the highest expense category with 40.44% and shopping is the lowest expense category with 3.68%.
- Mutual Funds is the best saving component and liquid cash is the least effective component for savings.
- December 2019 is the highest expense month and march 2018 is the highest saving month trends.
